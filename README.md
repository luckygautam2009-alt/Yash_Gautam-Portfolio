# Yash Gautam - AI & Web Developer Portfolio

A futuristic, highly interactive, and responsive portfolio website designed to showcase projects, achievements, skills, and carrier evolution. Built using React, TypeScript, Vite, TailwindCSS, and Framer Motion, with advanced glassmorphism design aesthetics and optimized for SEO and accessibility.

## 🚀 Live Demo
- [Live Portfolio Link]([https://yashgautam.dev/](https://yash-gautam-portfolio-v4xf.vercel.app/))

---

## 🛠️ Technology Stack
- **Frontend Core:** React 18, TypeScript, HTML5, Vanilla CSS
- **Styling:** TailwindCSS 4, Glassmorphic UI/UX tokens
- **Animations:** Motion (Framer Motion) for fluid UI/UX state transitions
- **Icons:** Lucide React
- **Build Tool:** Vite 6

---

## ✨ Features
1. **Interactive Terminal Hero:** An immersive terminal intro showing mock compilation and output execution of Yash's core identity in C.
2. **Career Evolution Timeline:** An interactive timeline tracing academic and professional milestones with floating pastel/glassmorphic skill pills.
3. **Interactive Project Wall:** Dynamic project pinboard representing agricultural AI systems, semantic segmentation models, and emergency coordination agents.
4. **Impact & Credentials Grid:** Achievements counters, certification galleries (AWS community day, full-stack certification), and open-source contribution metrics.
5. **Core Identity Constellation:** An interactive SVG-driven constellation node visualization representing Yash's developer personality and soft skill sets.
6. **Pastel Energy Orb Contact Section:** Glassmorphic contact cards supporting clean connection hooks to social ecosystems (LinkedIn, GitHub, Instagram, Mail).

---

## 📈 SEO & Accessibility Optimizations
The portfolio is optimized for search visibility and screen reader navigation, targeting a **95+ Lighthouse Score**:
- **Semantic HTML:** Corrected heading tags (single `<h1>` for page title, sections mapped to `<h2>`).
- **Screen Reader Friendly:** Added descriptive `aria-label` tags to floating navigation docks, interactive buttons, and external hyperlinks.
- **Dynamic Alt Attributes:** Modified raw image components to use contextual, descriptive alt texts instead of generic keywords.
- **Search Metadata:** Configured Open Graph (OG), Twitter Card meta tags, canonical link, author keywords, and JSON-LD structured data mapping profile details.
- **Indexation Tools:** Built a custom `robots.txt` configuration and XML `sitemap.xml` generated at the root build output.

## ⚙️ Local Development Setup

To run the development server locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/luckygautam2009-alt/Yash_Gautam-Portfolio.git
   cd Yash_Gautam-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
